== Installation ==

####Thank you for your interest in Spider Video [WordPress Player](https://web-dorado.com/products/wordpress-player.html).

Minimum requirements:  

* Wordpress 3.0+  
* PHP 5.x  
* MySQL 5.x  

After downloading the ZIP file  
1.  Login to your WordPress site administrator panel and head over the 'Plugins' menu  
2.  Click 'Add New'  
3.  Choose the 'Upload' option    
4.  Click **Choose file** (**Browse**) and select the Spider Video Player zip file.  
*For Mac Users*     
*Go to your Downloads folder and locate the folder with the player plugin. Right-click on the folder and select Compress. Now you have a newly created .zip file which can be installed as described here.*   
5.  Click **Install Now** button.  
6.  Once it is complete, activate the plugin.  

Once the plugin is activated, you'll notice a menu on the left called "Video Player".  
If any problem occurs with Spider [WordPress Video Plugin](http://wordpress.org/extend/plugins/player/), please contact us info@web-dorado.com.  


== Frequently Asked Questions ==

= Can I add Tags to Spider Video Player? =
  
The Tag is a custom parameter of videos by which you can search videos while creating playlists. Here are some examples of Tags:

  * Year,
  * Date,
  * Artist,
  * Album,
  * Genre,
  * Location,
  * Country,
  * Section.
  
On the left menu select Video Player > Tags.
In the upper left corner click on the "Add Tag" button for adding tags to Spider Video Player.

  * Name. Define a name for the tag for a Spider Video Player (e.g. Artist).
  * Required. Choose if the tag has to be defined for each video added or not in Spider Video Player.
  * Published. Choose whether to publish the tag or not in Spider Video Player.

= How can I adding videos to Spider Video Player? =
  
On the left menu select Video Player > Videos.
In the upper left corner click on the "Add video" button to Spider Video Player.

Title. Add a title for the video.
Type:

  * HTTP: Select the video (URL) and its HD version (UrlHD). You can provide the URL of the video or select it from the uploaded videos of Spider Video Player.  
  * YouTube: Provide the YouTube link to the video in the URL field.  
  * RTMP: Fill in the Flash Media Server (FMS) URL of the video along with the regular URL and UrlHD (HD version).  

Thumb. Select an image for the video thumbnail out of the uploaded images. Use the "Remove Image" option to remove the selected image.  
Tags. Fill in the available tags for your video in Spider Video Player (see Step 2 for adding tags).  

= What content does the Spider Video Player Flash option support? =

The Spider Video Player Flash options supports playback of any media content format supported by the Adobe Flash Player. This includes FLV, H.264/MPEG-4, MP3, AAC, JPG, PNG and GIF files, as well as XML playlists.

HTTP and RTMP streaming are supported, too.

= Why should I use Priority feature? =

The option is used for setting the priority of the Video player format.

If you choose Flash priority the videos will work with Flash in case of devices/browsers with Flash support, otherwise the HTML5 player will be used.

= How do I create playlists in Spider Video Player? =

On the left menu select  Video Player > Playlists.
In the upper left corner click on the "Add playlist" button.

  * Title. Choose a title for the playlist in Spider Video Player.
  * Videos. Use the "Add" button to add videos to your playlist from the list of existing videos.
  * Thumb. Select a thumbnail image for the playlist. Use the "Remove Image" option to remove the selected image.

= Is it possible to modify the appearance of videos or players? =

You can do that by creating/editing Themes of Spider Video Player plugin. On the left menu select  Video Player > Themes.

In the upper right corner click on the Add a Theme button. The themes are divided into the following sections:

  * General Parameters
  * Style Parameters
  * Playback Parameters
  * Playlist and Library Parameters
  * Video Control Parameters

= How can I add a player with the plugin? =

On the left menu select. Video Player > Spider Video Player.   
In the upper left corner click on the "Add player" button.

  * Priority. Choose if you want to add a Flash or HTML5 player. Note that if the device (e.g. mobile phone) does not support Flash, the player will automatically switch to HTML5 mode (videos of unsupported formats will not be displayed).
  * Title. Choose a title for the player.
  * Select Theme. Select a theme from the existing list.
  * Playlist. Use the "Add" button to add playlist to your player.

= How do I publish single created player? =

You can insert players into pages or posts the following way.

Press the button named Insert Spider Video Player in a post or a page you want to show the player.
Select a player from the list and click on the Insert button.

**Single Video**

  * Select a Video. Select the video you want to publish.
  * Select a Theme. Select a theme to be applied to the player.
  * Priority. Choose if you want to add a Flash or HTML5 player. Note that if the device (e.g. mobile phone) does not support Flash, the player will automatically switch to HTML5 mode (videos of unsupported formats will not be displayed).

Click on the Insert button.
Save the page or post.

Alternatively, you can use Spider Video Player without the button on the toolbar: Add the shortcode [Spider_Video_Player id="N"] (where N is the id of the player, for the player with id=2, the shortcode must be [Spider_Video_Player id="2"]) to the pages and posts, where you want the player to appear.

For a single video, the shortcode is the following: [Spider_Single_Video track="Video ID" theme_id="Theme ID" priority="0/1"], where Video ID should be replaced with the ID number of the video and Theme ID should be replaced with the ID number of the theme. The value of the priority parameter can be either "0" for Flash or "1" for HTML5 player.

= Is it possible to translate the Spider Video Player into another language? =

It is only possible to translate the front end of the plugin. To do so, you need to set the language of your website using Site Language option on WordPress Settings > General page. The plugin will be translated automatically.

You can also modify the translations using POEdit software. All translation files of the product are located under /languages folder of its directory.

You will need to download .po and .mo files of your language from here and modify .po with POEdit. Then please upload new .po and .mo back to the same folder, replacing their old versions.
